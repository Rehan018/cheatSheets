## Terminal ninja Techniques
### share files on Network using Python

```
python -m http.server port
```

### ngrok

```
ngrok http -auth="name:pass" file:///C:\Users\Seraj\Desktop
```

### copy output to clipboard

```
dir | clip
```

### Store output in .txt file

```
dir > path/name.txt
```
